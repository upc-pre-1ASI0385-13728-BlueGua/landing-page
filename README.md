# BlueGua – Landing Page

Proyecto del curso **(IHC – 1ASI0385)**.  
Desarrollo de una landing page informativa realizada con **HTML, CSS y JavaScript** sin frameworks.

---

## Descripción
**BlueGua** es una landing page que promueve el uso responsable del agua y la sostenibilidad.  
Incluye secciones informativas, un formulario de contacto y un diseño responsivo aplicando la guía de estilos del proyecto.
